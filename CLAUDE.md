# crow-frontend

Next.js 16 + React 19 기반 개인 가계부 프론트엔드.

## 하네스: Next.js 파이프라인

**목표:** product_manager → ui_architect → frontend_dev → [state_manager ‖ api_integrator] → [qa ‖ reviewer] 파이프라인으로 Next.js 기능을 엔드투엔드 구현

**트리거:** 새 화면/컴포넌트 추가, 기능 구현, 기존 UI 수정 등 프론트엔드 개발 작업 요청 시 `nextjs-orchestrator` 스킬을 사용하라. 단순 코드 설명이나 짧은 질문은 직접 응답 가능.
